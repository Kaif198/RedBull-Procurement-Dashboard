# Red Bull Procurement Dashboard

A comprehensive Red Bull Procurement Dashboard built with React and Vite.

## Features
- Simulator
- Risk Monitor
- Charts with Narratives
- Interactive Sidebar

## Getting Started

### Prerequisites
- Node.js

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Kaif198/RedBull-Procurement-Dashboard.git
   ```
2. Navigate to the project directory:
   ```bash
   cd RedBull-Procurement-Dashboard
   # If you have a frontend directory:
   cd frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
